  # DataMiningIA_BryanGuerrero

## Proyecto: Minería de Datos con Inteligencia Artificial

### Objetivo

Analizar la relación entre las horas de estudio y el rendimiento académico de los estudiantes mediante técnicas de minería de datos utilizando Excel y apoyo de inteligencia artificial.

### Pregunta de Investigación

¿Existe una relación entre las horas de estudio y las calificaciones obtenidas por los estudiantes?

### Hipótesis

Los estudiantes que dedican más tiempo al estudio tienden a obtener mejores calificaciones.

## Fuente de los Datos

Los datos fueron elaborados con fines académicos para analizar la relación entre las horas de estudio y el rendimiento académico.

### Variables Analizadas

- Estudiante
- Horas de estudio
- Nota final

## Proceso de Limpieza de Datos

Se realizó una revisión de los datos para:

- Verificar valores faltantes.
- Comprobar la ausencia de registros duplicados.
- Validar formatos numéricos.
- Organizar la información para su análisis en Excel.

## Técnica de Minería de Datos Aplicada

### Regresión Lineal

Se utilizó un gráfico de dispersión en Excel para identificar la relación entre las horas de estudio y las calificaciones obtenidas.

## Herramientas Utilizadas

- Microsoft Excel
- ChatGPT
- GitHub

## Resultados

El análisis mostró una tendencia positiva entre las horas de estudio y las calificaciones.

Los estudiantes que estudian más horas suelen obtener mejores resultados académicos.

## Conclusiones

1. Existe una relación positiva entre las horas de estudio y las calificaciones.
2. El tiempo dedicado al estudio influye en el rendimiento académico.
3. La minería de datos permitió identificar patrones relevantes dentro de los datos analizados.

## Limitaciones

- El conjunto de datos es pequeño.
- Los datos fueron creados con fines académicos.
- No se analizaron factores externos como aspectos económicos o familiares.

## Recomendaciones Futuras

- Utilizar una muestra más grande.
- Incorporar más variables relacionadas con el rendimiento académico.
- Aplicar técnicas de minería de datos más avanzadas.

## Uso de la Inteligencia Artificial

La inteligencia artificial fue utilizada para:

- Definir la pregunta de investigación.
- Formular la hipótesis.
- Seleccionar la técnica de análisis.
- Interpretar los resultados.
- Elaborar las conclusiones del proyecto.
